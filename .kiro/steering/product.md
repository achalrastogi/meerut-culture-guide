# Product Overview

## Meerut Culture Guide

A frontend-only cultural knowledge platform that uses AI-powered natural language processing (Google Gemini & OpenAI GPT) to help users understand Meerut city's rich cultural heritage, traditions, and local customs. The application serves as a cultural bridge, providing insights about Meerut's unique blend of ancient traditions and modern development.

## Core Purpose

- **Cultural Education**: Help users learn about Meerut's festivals, food culture, industries, and historical significance through natural language queries
- **Local Understanding**: Provide context about social nuances, customs, and religious harmony in Meerut
- **Interactive Learning**: Enable conversational queries with intelligent, contextual responses powered by AI
- **Honest Communication**: Clearly state when information is not available in the knowledge base

## Key Features

- **Dual AI Integration**: Choose between Google Gemini (recommended, free tier) or OpenAI GPT
- **Knowledge Base Driven**: All answers sourced from comprehensive cultural knowledge base (product.md)
- **Formatted Responses**: Beautiful formatting with numbered lists, bullet points, and bold text
- **Persistent Settings**: API keys saved in browser localStorage - configure once, use forever
- **Confidence Indicators**: Shows confidence level (High/Medium/Low/None) for each response
- **Source Attribution**: References specific sections of the knowledge base used
- **Honest Responses**: Clearly states when information is not available
- **Modern Chat UI**: Clean, responsive interface with sidebar navigation and example questions
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Target Users

- Visitors to Meerut seeking cultural understanding
- Researchers studying regional Indian culture
- Business professionals working in Meerut
- Students learning about local traditions and customs
- Anyone curious about Meerut's rich heritage

## Cultural Focus Areas

- **Historical Heritage**: 1857 Rebellion, Vedic period, Buddhist heritage, ancient origins
- **Festivals & Celebrations**: Nauchandi Mela, Dussehra, Ganga Aarti, religious harmony
- **Food Culture & Specialties**: Gazak, Revri, Halwa Paratha, street food traditions
- **Local Industries**: Sports goods manufacturing, brass work, textile craftsmanship
- **Social Customs**: Wedding traditions, hospitality, community gatherings
- **Modern Culture**: Educational institutions, youth culture, contemporary developments
- **Tourist Places**: Augharnath Temple, St. John's Church, Gandhi Bagh, Shahid Smarak, Suraj Kund

## Technical Approach

- **Frontend-Only Architecture**: No backend server required - pure static site
- **Static Knowledge Base**: Cultural information stored in markdown file (product.md)
- **AI-Powered Processing**: Google Gemini or OpenAI GPT for natural language understanding
- **Client-Side Storage**: API keys stored in browser localStorage for persistence
- **Privacy-First**: API keys never sent to our servers, only to Google/OpenAI directly
- **GitHub Pages Deployment**: Automatic deployment via GitHub Actions
- **Fast & Reliable**: Instant responses with AI-powered understanding

## User Experience

- **Chat Interface**: Modern conversational UI with message history
- **Category Navigation**: Browse by festivals, food, history, industries, customs
- **Popular Topics**: Quick access to frequently asked questions
- **Example Questions**: Pre-populated questions to get started quickly
- **Settings Panel**: Easy configuration of AI provider and API key
- **Test Connection**: Verify API key before using
- **Formatted Output**: Responses with proper lists, bold text, and paragraphs
